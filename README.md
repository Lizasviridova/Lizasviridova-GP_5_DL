# Описание и бизнес задача

**Цель**

Основная наша цель это построить модели для предсказания стоимости дома, квартиры, жилья по разным признакам. 

**Зачем?**

Наша задача может помочь многим платформам для поиска жилья предсказывать и оценивать стоимость жилья.

**За что отвечает каждая часть проекта**

Первая часть нашего проекта прогнозирует стоимость объекта по табличным характеристикам, а вторая часть анализирует фотографии интерьера.

Подробнее про вторую часть: 
мы автоматически определяем интерьерный стиль объекта по фотографиям, добавляем теги к объявлению, улучшаем фильтры и рекомендации покупателям.

Автоматическое определение стиля объекта по его фотографии так же может улучшить предсказание цены объекта (дома, здания, квартиры и тд). Сейчас цена помещаения, дома, квартиры и тд. может зависить от многих факторов, к примеру внешний вид помещения, внутренний вид, его стиль.

# Данные

Для 1 части: 
Датасет kc_house_data_NaN.csv - продажи домов в округе Кинг (США).

Основные поля: дата продажи, цена, количество спален и ванных, площадь, этажность, год постройки, zipcode и другие признаки. Задача - предсказать цену по остальным признакам.

Подробнее:

id - уникальный идентификатор транзакции

date - дата продажи

price - цена продажи (целевая переменная)

bedrooms - количество спален

bathrooms - количество ванных комнат

sqft_living - жилая площадь

sqft_lot - площадь участка

floors - количество этажей

waterfront - наличие вида на набережную

view - качество вида (оценка 0-4)

condition - состояние дома (оценка 1-5)

grade - общий рейтинг дома (оценка 1-13)

sqft_above - площадь надземной части

sqft_basement - площадь подвала

yr_built - год постройки

yr_renovated - год реновации

zipcode - почтовый индекс

lat - широта

lon - долгота

sqft_living15 - средняя жилая площадь 15 ближайших соседей

sqft_lot15 - средняя площадь участка 15 ближайших соседей

Для 2 части:
Architectural Styles Periods Dataset - интерьерные стили домов

Какие данные: сами стили и подборка изображений под каждый из них
